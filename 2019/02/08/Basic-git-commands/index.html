How to use git init,git add,git commit-v

  1. git init - 初始化仓库
    Git使用git init命令初始化仓库。
    首先建立一个空的目录，初始化成功，该目录下就会生成一个.git目录。.git目录中存放着管理当前目录所需的仓库数据。

  1. git add - 向暂存区添加文件
    要想让文件成为Git的管理对象，需要通过git add 命令将其加入暂存区。
    暂存区是提交前的一个临时区域。

  1. git commit -v :提交时显示所有diff信息
    输入以上命令，再按回车，就可以看到所有的diff信息

  2. 如果遇到了别的不懂的命令，可以使用explainshell这个网站来查询解释
    explainshell.com
