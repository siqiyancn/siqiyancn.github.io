How to use ls,cat,mv,touch

  1. ls
    Is是list的缩写，表示显示文件目录列表。（只显示非隐藏文件的文件名）
    一些用法:
    -a：–all的缩写，显示所有文件，包括隐藏文件（以.开头的文件）
    -L：列出长数据串，显示文件的大小，时间等数据信息。
    ls -la：表示显示文件包括的所有信息。
  2. cat
    主要有三个功能：

一次显示整个文件：cat filename
从键盘创建一个文件：cat > filename 只能创建新文件，不能编辑已有文件。
将几个文件合并为一个文件：cat file1 file2 > file

  1. mv
    mv是move的缩写,表示重命名或者移动文件。
    命令格式：mv 源文件 目标文件

  2. touch
    创建文件,改变文件更新时间
    命令格式： touch file
    若file不存在，则创建新文件，存在，就修改系统时间

如果遇到了别的不懂的命令，可以使用explainshell这个网站来查询解释
explainshell.com

How to use git init,git add,git commit-v

  1. git init - 初始化仓库
    Git使用git init命令初始化仓库。
    首先建立一个空的目录，初始化成功，该目录下就会生成一个.git目录。.git目录中存放着管理当前目录所需的仓库数据。

  1. git add - 向暂存区添加文件
    要想让文件成为Git的管理对象，需要通过git add 命令将其加入暂存区。
    暂存区是提交前的一个临时区域。

  1. git commit -v :提交时显示所有diff信息
    输入以上命令，再按回车，就可以看到所有的diff信息

  2. 如果遇到了别的不懂的命令，可以使用explainshell这个网站来查询解释
    explainshell.com
